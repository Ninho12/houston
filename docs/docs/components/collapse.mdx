### Propriedades disponíveis

| prop           | tipo       | obrigatório | padrão | descrição                                                           |
| -------------- | ---------- | ----------- | ------ | ------------------------------------------------------------------- |
| visibled       | `boolean`  | true        | -      | -                                                                   |
| onEnter        | `function` | false       | -      | Função chamada ao clicar para expandir o conteúdo.                  |
| destroyOnClose | `boolean`  | false       | false  | Se verdadeiro, ao ocultar o conteúdo remove o `children` do render. |
