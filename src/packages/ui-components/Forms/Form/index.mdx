# Form

Componente de formulário de alto desempenho com gerenciamento de escopo de dados.

### Importação

```js
import Form from '@eduzz/houston-ui/Forms/Form';
```

### Exemplo

Exemplo de uso [aqui](https://eduzz.github.io/houston/form/example).

### Contexto interno

Utilizado internamente no componente `<Form />`. Recomendado o uso apenas para compartilhar o estado do formulário com os componentes filhos.

```js
import Form, { useFormContext } from '@eduzz/houston-ui/Forms/Form';

const Component = () => {
  return (
    <Form {...props}>
      <SubComponent />
    </Form>
  );
}

const SubComponent = () => {
  const form = useFormContext();
  ...
};
```

### Propriedades

| prop      | tipo                | obrigatório | padrão | descrição |
| --------- | ------------------- | ----------- | ------ | --------- |
| id        | `string`            | `false`     | -      |           |
| className | `string`            | `false`     | -      |           |
| context   | `IFormAdapter<any>` | `true`      | -      | -         |
