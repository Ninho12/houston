# Tabela

Tabelas apresentam informações de uma forma fácil de visualizar, de modo que os usuários podem
procurar por padrões e percepções.

### Importação

```js
import Table from '@eduzz/houston-ui/Table';
```

## Propriedades disponíveis

### Table props

| prop         | tipo             | obrigatório | padrão   | descrição                                                                               |
| ------------ | ---------------- | ----------- | -------- | --------------------------------------------------------------------------------------- |
| loading      | `boolean`        | `false`     | `false`  | -                                                                                       |
| onSortable   | `function`       | `false`     | -        | Função chamada ao clicar em uma coluna `sortable`.                                      |
| stickyHeader | `boolean`        | `false`     | `false`  | Header fixo, é necessário inserir um valor na prop `maxHeight` para melhor experiência. |
| size         | `Size`           | `false`     | `medium` | -                                                                                       |
| maxHeight    | `number`         | `false`     | -        | -                                                                                       |
| messages     | `ITableMessages` | `false`     | `false`  | Mensagens para eventos internos.                                                        |

### Table.Column props

| prop     | tipo              | obrigatório | padrão  | descrição                                                                                          |
| -------- | ----------------- | ----------- | ------- | -------------------------------------------------------------------------------------------------- |
| field    | `string`          | `true`      | -       | Chave única.                                                                                       |
| label    | `React.ReactNode` | `true`      | -       | -                                                                                                  |
| width    | `number`          | `false`     | -       | -                                                                                                  |
| align    | `ITableAlign`     | `false`     | `left`  | -                                                                                                  |
| sortable | `boolean`         | `false`     | `false` | Se `true`, ao clicar na coluna, chama a função `onSortable` passando a prop `field` e a ordenação. |

### Table.Row props

| prop | tipo      | obrigatório | padrão | descrição                                                                     |
| ---- | --------- | ----------- | ------ | ----------------------------------------------------------------------------- |
| data | `unknown` | `true`      | -      | Valores da linha, é necessário apenas quando há o componente `Table.Actions`. |

### Table.Cell props

| prop     | tipo           | obrigatório | padrão         | descrição                                                   |
| -------- | -------------- | ----------- | -------------- | ----------------------------------------------------------- |
| align    | `ITableAlign`  | `false`     | `left`         | -                                                           |
| truncate | `number`       | `false`     | -              | Quando preenchido, limita o valor do texto (se for string). |
| colSpan  | `number`       | `false`     | -              | -                                                           |
| xs       | `ITableCellXs` | `false`     | `{ size: 12 }` | Controle de tamanho no mobile.                              |

### Table.Actions props

| prop  | tipo                       | obrigatório | padrão  | descrição |
| ----- | -------------------------- | ----------- | ------- | --------- |
| label | `React.ReactNode`, `false` | `false`     | `Ações` | -         |
| align | `ITableAlign`              | `false`     | `left`  |           |

### Table.Options props

| prop     | tipo                  | obrigatório | padrão  | descrição                           |
| -------- | --------------------- | ----------- | ------- | ----------------------------------- |
| onClick  | `function`            | `false`     | -       | -                                   |
| disabled | `function`, `boolean` | `false`     | `false` | -                                   |
| hide     | `function`, `boolean` | `false`     | `false` | Controla a rederização do elemento. |
| icon     | `React.ReactNode`     | `false`     | -       | -                                   |

### Table.Pagination props

| prop                | tipo              | obrigatório | padrão                  | descrição                                                 |
| ------------------- | ----------------- | ----------- | ----------------------- | --------------------------------------------------------- |
| page                | `number`          | `true`      | -                       | -                                                         |
| perPage             | `number`          | `true`      | -                       | -                                                         |
| totalPages          | `number`          | `true`      | -                       | -                                                         |
| optionsPerPage      | `number[]`        | `false`     | `[15, 25, 35, 50, 100]` | -                                                         |
| labelItensPerPage   | `React.ReactNode` | `false`     | `Itens por página:`     | -                                                         |
| onChangeRowsPerPage | `function`        | `false`     | -                       | Se a prop não existir, o componente não será renderizado. |
| onChangeGoToPage    | `function`        | `false`     | -                       | Se a prop não existir, o componente não será renderizado. |
| onChangePage        | `function`        | `false`     | -                       | Se a prop não existir, o componente não será renderizado. |

### Table.Collapse props

| prop       | tipo            | obrigatório | padrão  | descrição                                                      |
| ---------- | --------------- | ----------- | ------- | -------------------------------------------------------------- |
| loading    | `boolean`       | `false`     | -       | -                                                              |
| onCollapse | `function`      | `false`     | -       | Função chamada ao clicar para abrir / fechar o item recolhido. |
| type       | `list`, `table` | `false`     | `table` | Se `list` não renderizará as colunas e bordas inferiores.      |
