# Date Picker Field

O componente é responsável por exibir um campo de data nos formulários.

### Importação

```js
import DatePickerField from '@eduzz/houston-ui/Forms/DatePicker';
```

### Exemplo

<Playground>
  {() => {
    const [value, setValue] = React.useState(null);
    return (
      <>
        <DatePickerField width={200} label='Data' value={value} onChange={value => setValue(value)} />
        <DatePickerField
          width={200}
          label='Data'
          placeholder='Insira a data'
          value={value}
          onChange={value => setValue(value)}
        />
      </>
    );
  }}
</Playground>
