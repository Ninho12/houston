# Color Field

O componente é responsável por exibir um campo de seleção de cor hexadecimal.

### Importação

```js
import ColorField from '@eduzz/houston-ui/Forms/Color';
```

### Exemplos

<Playground>
  {() => {
    const [color, setColor] = React.useState('');
    return (
      <>
        <ColorField label='Cor' value={color} onChange={setColor} />
        <ColorField label='Cor' placeholder='Selecione uma cor' value={color} onChange={setColor} />
        <ColorField label='Cor padrão' defaultValue='#4a8b90' value={color} onChange={setColor} />
        <ColorField label='Campo desabilitado' value={color} onChange={setColor} disabled />
      </>
    );
  }}
</Playground>

### Propriedades

| prop           | tipo                          | obrigatório | padrão  | descrição                                                                           |
| -------------- | ----------------------------- | ----------- | ------- | ----------------------------------------------------------------------------------- |
| name           | `string`                      | `true`      | -       | -                                                                                   |
| label          | `React.ReactNode`             | `false`     | -       | -                                                                                   |
| placeholder    | `string`                      | `false`     | -       | -                                                                                   |
| disabled       | `boolean`                     | `false`     | `false` | -                                                                                   |
| fullWidth      | `boolean`                     | `false`     | `true`  | -                                                                                   |
| loading        | `boolean`                     | `false`     | -       | Exibe ícone de carregamento no componente de texto.                                 |
| helperText     | `string`                      | `false`     | -       | Exibe texto de ajuda abaixo do componente de texto.                                 |
| errorMessage   | `string`                      | `false`     | -       | Exibe uma mensagem de erro no componente de texto, assim como indica erro no campo. |
| value          | `string`                      | `false`     | -       | -                                                                                   |
| defaultValue   | `string`                      | `false`     | -       | Valor padrão do campo.                                                              |
| margin         | `none`, `dense`, `normal`     | `normal`    | -       | -                                                                                   |
| onChange       | `function`                    | `false`     | -       | Função disparada ao alterar o valor do campo de texto.                              |
| endAdornment   | `React.ReactNode`             | `false`     | -       | Ícone ou botão com ícone que será exibido no final do campo.                        |
| startAdornment | `React.ReactNode`             | `false`     | -       | Ícone ou botão com ícone que será exibido no início do campo.                       |
| required       | `boolean`                     | `false`     | -       | -                                                                                   |
| InputProps     | `Partial<OutlinedInputProps>` | `false`     | -       | Props aplicadas no input.                                                           |
