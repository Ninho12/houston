# Houston UI Components

[![version](https://img.shields.io/npm/v/@eduzz/houston-ui)](https://www.npmjs.com/package/@eduzz/houston-ui)

## Instalação

```bash
yarn add @eduzz/houston
```

Não é necessário nenhuma configuração a mais para usar, caso queria customizar as cores e o temas veja
o menu [Temas e Estilos](/ui-components/theme)

## Como usar

```tsx
import Button from '@eduzz/houston-ui/Button';

const MyComponent = () => {
  return <Button>Meu Botão</Button>;
};
```
