# Instalação

Para instalar o Houston basta adicionar cada package:

```bash
yarn add @eduzz/houston-ui @eduzz/houston-forms @eduzz/houston-hooks
```

Para configurar no padrão do Houston no eslint, você deve adicionar as dependências, pois o eslint exige que plugins devem,
ser adicionados no projeto principal:

```bash
yarn add --dev @eduzz/eslint-config-houston @typescript-eslint/eslint-plugin @typescript-eslint/parser eslint-config-prettier eslint-plugin-eslint-plugin eslint-plugin-import eslint-plugin-prettier eslint-plugin-react eslint-plugin-react-hooks eslint-plugin-unused-imports prettier eslint-plugin-sonarjs eslint
```

Veja mais sobre as configurações no menu [ESLint](/eslint).
