## Propriedades da Table.Column

Usando tabela do próprio markdown

| prop     | tipo              | obrigatório | padrão  | descrição                                                                                    |
| -------- | ----------------- | ----------- | ------- | -------------------------------------------------------------------------------------------- |
| field    | `string`          | true        | -       | Chave única.                                                                                 |
| label    | `React.ReactNode` | true        | -       | -                                                                                            |
| width    | `number`          | false       | -       | -                                                                                            |
| align    | `ITableAlign`     | false       | `left`  | -                                                                                            |
| sortable | `boolean`         | false       | `false` | Quando `true`, ao clicar na coluna, chama a função` onSortable` passando o `field` e o sort. |
